Features

Works with All your favorite libraries

React-Sight supports React 14.7+ (including Fiber), React Router v4, and React Redux

neatly packaged

React-Sight is a light-weight, unobtrusive module that will show you how your Application is working

See only what you want to see

Built in filters allow you to hide elements you don't want to see, allowing you focus only on the components that you've written
